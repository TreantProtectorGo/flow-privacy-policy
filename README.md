# Flow Privacy Policy

Static site for the public privacy policy of `Flow`.

## Publish With GitHub Pages

1. Create a GitHub repository named `flow-privacy-policy`.
2. Push this directory to the `main` branch.
3. In the repository settings, enable GitHub Pages from the `main` branch root.
4. Use the published URL as your App Store Connect `Privacy Policy URL`.

Expected Pages URL:

`https://<your-github-username>.github.io/flow-privacy-policy/`
